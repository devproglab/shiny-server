My name is Ilya Gulenkov and I am a 2021 graduate student of HSE (Moscow). As I plan to continue working in research-related area, this website will serve as a live portfolio reflecting my current projects.

An alpha version of the dashboard which decomposes product export diversification for 130+ countries using the Theil index. It tracks export diversification decomposition along the intensive and extensive marings for 2001 - 2019. The index is calculated using the IMF (2014, Annex 2) approach and employs TradeMap (2021) data
